  <Steps>
    <Step title="Invite the bot to your server">
      Add AutoResponse to your Discord server using the invite link. Make sure the bot has permissions to read messages and send replies.
    </Step>
    
    <Step title="Configure reply channels">
      Use the `/addreply channel` command to enable auto-replies in specific channels. The bot will start with a 6% reply chance that increases with each message.
      
      ```bash
      /addreply channel #general
      ```
    </Step>
    
    <Step title="Add custom phrases">
      Create personalized reply phrases using `/addphrase`. The bot will randomly select from your phrases when it triggers a reply.
      
      ```bash
      /addphrase "Welcome to the server! 🎉"
      ```
    </Step>
    
    <Step title="Track the leaderboard">
      Check who's getting the most auto-replies with `/leaderboard`. Compete with your community for the top spot!
      
      ```bash
      /leaderboard season:Current
      ```
    </Step>
  </Steps>
</div>
